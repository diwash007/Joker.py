# Joker.py


## About

A simple python package to get jokes and puns.

## How to use?

You can use this by package from [Joker.py]()
```sh
pip install joker.py
```

## Why use this?

To fetch jokes and puns on your projects.

## How to use?

```py
from joker import Joker


joker = Joker()
joke = joker.joke()
print(joke)

puns = joker.puns(5)
for pun in puns:
    print(pun)

```
